# ChatUp – Realtime Messaging Platform

ChatUp is a cross‑platform Flutter application that provides real‑time messaging with a clean, modern UI and Firebase‑backed data flow. The project includes authentication, user discovery, and one‑to‑one chat capabilities with message UI components built for responsiveness across mobile and web.

## Project Description

This project is a real‑time messaging platform built with Flutter and Firebase. It focuses on delivering a smooth chat experience with secure authentication, user discovery, and responsive chat interfaces. The codebase is organized into clear layers for API access, data models, screens, and reusable widgets, making it easy to maintain and extend. The application targets multiple platforms (Android, iOS, Web, and desktop) from a single codebase.

## Highlights

- Realtime chat experience
- Firebase integration (Auth + data layer)
- Modular, maintainable architecture (API, Models, Screens, Widgets)
- Responsive UI components for chat and user lists
- Multi‑platform support (Android, iOS, Web, Windows, macOS, Linux)

## Tech Stack

- Flutter (Dart)
- Firebase (Auth, Realtime data/Firestore)

## Project Structure

- lib/main.dart – App entry point and initialization
- lib/API/_Apis.dart – Backend/Firebase API integration
- lib/Models/Message.dart – Message model
- lib/Models/Chat_user.dart – User model
- lib/Screens/ – UI screens (home, chat, authentication, splash)
- lib/Widgets/ – Reusable UI components

## Getting Started

### Prerequisites

- Flutter SDK installed
- Firebase project configured (web/android/ios as needed)

### Install Dependencies

1. Open a terminal in the project root
2. Run flutter pub get

### Run the App

- Android/iOS: flutter run
- Web: flutter run -d chrome

## Configuration

If you are setting up Firebase, ensure your platform config files are present:

- android/app/google-services.json
- ios/Runner/GoogleService-Info.plist
- lib/firebase_options.dart (generated by FlutterFire CLI)
